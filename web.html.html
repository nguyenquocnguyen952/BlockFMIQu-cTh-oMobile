Hướng dẫn

Cách cài đặt trên iPhone

1

Nhấn nút tải file bên trên

Trình duyệt Safari sẽ tải file .mobileconfig về thiết bị. Nên dùng Safari để tránh lỗi.

2

Mở Cài đặt → Đã tải về Profile

Vào Settings (Cài đặt) → sẽ xuất hiện mục "Đã tải về Profile" ở trên cùng. Nhấn vào đó.

3

Nhấn Cài đặt → Tiếp tục → Cài đặt

Xác nhận cài đặt profile. Nhập mật khẩu màn hình nếu được yêu cầu. Nhấn Cài đặt để hoàn tất.

4

Khởi động lại thiết bị

Restart iPhone để profile được áp dụng hoàn toàn. Sau đó thiết bị sẽ chặn kết nối iCloud tự động.
